# 📌 Projects

## Project A - Advanced Scientific Calculator

A command-line calculator developed using Python.

### Features
- Addition
- Subtraction
- Multiplication
- Division
- Power
- Square Root
- Logarithm
- Natural Logarithm
- Factorial
- Sine, Cosine and Tangent
- Prime Number Check
- Input Validation using `try-except`

---

## Project B - Number Guessing Game

A Python game where the player guesses a randomly generated number.

### Features
- Easy, Medium and Hard difficulty levels
- Random number generation
- Limited attempts
- Best Score tracking
- Games Played, Won and Lost statistics
- Replay option
- Input validation
- Range validation

---

## Project C - Student Management System

A file-based Student Management System developed using Python.

### Features
- Add Student
- View Students
- Search Student
- Update Student Name & Grade
- Delete Student
- File Handling using `students.txt`
- Duplicate Roll Number validation
- Marks validation
- Menu-driven program
